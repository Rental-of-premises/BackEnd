1) Для запуска нужно установить docker и docker-compose чтобы работала bd

ps. это на много проще чем вручную настраивать бд

Нужно убедиться что всё работает. И docker-ы запущены 

Можно просто сразу запустить службы:
```bash
sudo systemctl start docker
```

2) Далее запускаем докер с бд в корневой дирректории проекта

```bash
sudo docker-compose up --build
```

Если всё нормально, база данных уже работает на порту 5432 со всеми необходимыми таблицами. Можно зайти через pgAdmin по порту 5050. Но там нужно вручную вбивать все данные сервера, которые есть в .example.env

Там можно попробовать добавить одного пользователя в таблицу users, чтобы проверить, что всё ок

3) Бд уже запущено, нужно запустить сервер (в отдельном терминале)

устанавливаем пакеты go

```bash
go mod download
```

запускаем, обязательно находясь в корневой дирректории проекта, чтобы подгрузился .env файл

```bash
go run ./cmd/rent/
```

программа должна запуститься, вывести строку localhost:3000 и не завершиться

4) чтобы проверить что работает, предварительно нужно добавить какого-то пользователя в bd

а далее в браузер вбиваем: http://localhost:3000/api/users/1

Если вывело пользователя, всё зашибись, работает
